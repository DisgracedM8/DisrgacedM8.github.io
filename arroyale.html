ARRoyale was a project for college to use AR and implement a game mechanic into it. My partner and I both played the game Clash Royale where you place troops down and they fight each other. We thought it would be cool if you take that gameplay loop and instead transfer that to be able to go down to their level and see it from a different point of view. The assignment was originally supposed to be 2D but we were set on trying our hand at 3D. It was also the first project I got to use Unity's navmesh and implement proper ai into a game. It currently only has a Google build as it has not been transferred to IOS and has to be downloaded by the apk. This gave me the experience of developing in AR, AI, and optimizing for mobile devices that do not have as many resources as a desktop.
